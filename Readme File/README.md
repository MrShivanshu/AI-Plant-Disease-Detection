# AI Plant Disease Detection 🌿

## Overview

This project is an AI-based plant disease detection system that identifies diseases in plant leaves using deep learning.

Users can upload a leaf image, and the system predicts the disease along with confidence score and treatment suggestions.

## Technologies Used

* Python
* TensorFlow / Keras
* EfficientNet-B0
* FastAPI
* HTML, CSS, JavaScript

## Dataset

PlantVillage Dataset

## Features

* Upload leaf image
* Detect plant disease
* Show confidence score
* Provide treatment suggestion

## Run Project

```bash
python -m uvicorn backend.main:app --reload
```

Open in browser:

```plaintext
http://127.0.0.1:8000/
```
